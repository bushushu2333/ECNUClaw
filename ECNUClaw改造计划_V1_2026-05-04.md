# ECNUClaw 改造计划

> 基于张治教授的个性化学习与学习者画像理论

## 一、理论融入方案

### 1. 学习者画像系统（对应张治「数字画像三层框架」）

| 张治的理论 | ECNUClaw 实现 | 当前状态 |
|-----------|-------------|---------|
| 指标体系层 | HEADS prompt 中定义画像维度（认知、行为、情感、元认知、情境） | 需要改造 memory.py + heads.py |
| 数据实践层 | 多源数据采集：对话行为、工具调用记录、回答正确率、情绪信号 | 需要新增数据采集层 |
| 数字画像层 | SQLite 持久化 5 维画像 + 可视化输出 | 需要扩展 memory.py |

**5 个画像维度：**
1. **认知维度** — 知识状态（掌握/部分/未掌握）、认知水平（布鲁姆分类）、先验知识
2. **行为维度** — 学习行为模式、参与度（提问频率、会话时长）、工具使用习惯
3. **情感维度** — 学习情绪（困惑/挫折/兴奋）、动机水平、自我效能感
4. **元认知维度** — 自我调节能力、学习策略偏好、反思能力
5. **情境维度** — 年级、学习环境、时间偏好

### 2. 教育大脑模型（对应张治「教育大脑三层架构」）

| 层级 | 功能 | ECNUClaw 模块 |
|-----|------|-------------|
| 类脑感知神经系统 | 多源数据采集 | CLI 交互层 + 意图路由 |
| 类脑中枢神经系统 | AI 分析 + 画像构建 + 决策 | Agent + Memory + Planner |
| 类脑运动神经系统 | 个性化输出 + 自适应干预 | HEADS Prompt + 工具调用 |

### 3. 人机协同智商（对应张治「ChatGPT/AIGC重塑教育」）

- 不直接给答案 → 引导思考 → 培养「与AI协作解决问题」的能力
- 系统设计目标：不是替代学习，而是增强学习者的思维能力和元认知

## 六、张治教授理论调研详情

### 核心著作
- *走进学校3.0时代* — 学校从1.0（传统）→ 2.0（信息化）→ 3.0（智能）的进化
- *数字画像-智能时代综合素质评价新范式* — 多源多维评价的系统性阐述
- *走进自适应学习时代* — 自适应学习的理论与实践

### 数字画像三层框架（2021）
1. **指标体系层** — 基于学术研究和政策文件，定义综合素质的维度
2. **数据实践层** — 多源数据收集和融合，处理行为、学业和其他数据流
3. **数字画像层** — 构建学生可视化、动态的画像

关键转变：从静态的考试分数驱动 → 动态的、数据驱动的、多维度画像

### 教育大脑模型（2022）
仿生计算架构，模仿大脑的三层神经系统：
- **类脑感知神经系统** — 数据采集层，捕获多模态教育数据（LMS、IoT设备、互动日志）
- **类脑中枢神经系统** — 处理层，使用AI、知识图谱和学习者画像进行数据处理、分析和决策
- **类脑运动神经系统** — 行动层，输出个性化推荐、自适应干预和管理仪表盘

### 人机协同智商（2023）
- 衡量人类与AI系统有效协作能力的新型智能
- 知识观转变：生成式AI将知识从静态信息转变为动态、可共创的资源
- 六个重塑领域：传统教育工具有效性、知识价值论、资源形态、学习模型、评价方法、人机协作

### 学习者画像的通用学术维度（文献综述）

| 维度 | 子维度 | 数据来源 |
|-----|--------|---------|
| 认知 | 知识状态、认知能力（布鲁姆）、先验知识、认知负荷 | 问答正确率、知识追踪 |
| 行为 | 学习行为模式、参与度、互动模式、节奏和坚持 | 点击流、会话日志、工具调用 |
| 情感 | 学习情绪、动机水平、自我效能感、态度和兴趣 | 情感计算、自我报告 |
| 元认知 | 自我调节、学习策略偏好、反思能力 | 策略选择、目标设定 |
| 情境 | 学习环境、时间语境、社会语境 | 环境传感器、时间戳 |
